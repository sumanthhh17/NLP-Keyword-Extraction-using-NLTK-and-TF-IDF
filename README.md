# NLP Keyword Extraction using NLTK and TF-IDF

A simple Natural Language Processing (NLP) project built with Python that demonstrates the complete text preprocessing pipeline and extracts important keywords using the TF-IDF (Term Frequency-Inverse Document Frequency) algorithm..

## 📌 Features

- Tokenizes input text into individual words
- Removes English stopwords
- Filters out punctuation and non-alphabetic tokens
- Reconstructs processed text
- Calculates TF-IDF scores
- Displays keywords with their TF-IDF scores
- Shows the top 5 most important keywords

## 🛠️ Technologies Used

- Python 3
- NLTK
- Scikit-learn


## 🔄 Workflow

1. Accept user input.
2. Tokenize the sentence using NLTK.
3. Remove stopwords and punctuation.
4. Convert filtered words back into a sentence.
5. Generate TF-IDF vectors using Scikit-learn.
6. Display TF-IDF scores.
7. Print the top five keywords.

## 📚 Libraries Used

### NLTK

- English Stopword Removal

### Scikit-learn
- TF-IDF Vectorizer
